Budgeting

Project Description

Budgeting application in Spring that allows users to manage their budget on a monthly basis.

Features

  • Simple login / logout
  • Users can add accounts
  • Users can add expenses
  • Users can view expenses
  • com.example.demo.User will be able to see balance at all times - Display top right / displayed on Navbar
  • com.example.demo.User will be able to delete / update accounts - such as removing an account / changing it
